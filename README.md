# ASUAndroid
ASU Application for Army Military Members

(1):
```
1.) Enable SVM in BIOS
2.) Enable Hypervisor
Windows:
https://docs.microsoft.com/en-us/virtualization/hyper-v-on-windows/quick-start/enable-hyper-v

iOS:
https://stackoverflow.com/questions/13580491/how-to-enable-support-of-cpu-virtualization-on-macbook-pro

Linux:
https://www.nakivo.com/blog/run-linux-hyper-v/
Note: You might have to configure the Bios on some machines

```
(2)
```
1.) Install Android Studio
2.) Restart Machine
```
(3)
```
1.) Launch android studio and click File 
                                    -> New  
                                      -> Import Project
2.) Navigate to the unzipped package that is downloaded on your system and select the Android icon with ‘ASU Android'
```
(4)
```
1.)	Open AVD manager and Create New Virtual Device by going to Tools
                                                              -> AVD Manager 
                                                                -> Create new Virtual Device
2.) Choose “Pixel 3a” Choose Next -> DownLoad System image required and then finish 
```

(5)
```
1.)	Launch your Virtual device by clicking green Play arrow at the top
```
